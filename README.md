# SQLite Warehouse Inventory Project

Учебный консольный проект на Python для складского учёта товаров с использованием SQLite.

## Возможности

* Показать все товары
* Добавить новый товар
* Проверить дубли перед добавлением
* Обновить остаток товара
* Удалить товар
* Показать товары с низким остатком
* Показать товары для дозаказа
* Безопасно принимать числовой ввод
* Хранить данные в SQLite-базе

## Структура проекта

```text
sqlite_inventory_project/
├── main.py
├── db_utils.py
├── product_utils.py
├── input_utils.py
├── warehouse.db
├── README.md
└── .gitignore
```

## Файлы проекта

* `main.py` — основное меню программы
* `db_utils.py` — функции для работы с SQLite
* `product_utils.py` — расчёт стоимости, статуса и печать товаров
* `input_utils.py` — безопасный ввод чисел
* `warehouse.db` — SQLite-база данных с товарами

## Как запустить

```bash
python main.py
```

## Меню программы

```text
1. Показать все товары
2. Добавить товар
3. Показать товары с низким остатком
4. Показать товары для дозаказа
5. Обновить остаток товара
6. Удалить товар
0. Выйти
```
